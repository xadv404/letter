Phase 3

Scoring SQLi

Chaque paramètre URL est noté via regex + wordlist SecLists Burp. Les URLs à risque sont flaggées.

Tier Score Exemples Verdict
HIGH ≥ 85 id search_term filter_by Accepté · URL flaggée
MEDIUM 65 – 84 SecLists match · param inconnu plausible Accepté pour dorks
LOW 50 – 64 page sort category Exclu par défaut
EXCLUDE < 50 utm_source · tokens session Ignoré 
/products?category=electronics&id=42

category  →  LOW (58)   ✗ rejeté
id        →  HIGH (95)  ✓ targets_*.txt
